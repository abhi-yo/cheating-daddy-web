Clear Storage Guide

Learn how to clear Cheating Daddy's storage and reset all app data.
This will remove settings, cache, and any stored information.

⚠️ Warning: Clearing storage will remove all your app settings, saved configurations, and cached data. You'll need to reconfigure the app after doing this.
1

Close Cheating Daddy

Before clearing storage, make sure Cheating Daddy is completely closed. Exit the app from the system tray or force quit it if necessary.
2

Select Your Platform

Windows
macOS

For Windows Users:

You need to delete the following folders:

Primary storage location:
C:\Users\<YourUsername>\AppData\Local\cheating-daddy
Also check the Roaming folder:
C:\Users\<YourUsername>\AppData\Roaming\cheating-daddy

For macOS Users:

Delete the following folder:

~/Library/Application Support/cheating-daddy
3

Delete the Folder

Once you've located the "cheating-daddy" folder(s), right-click and delete them. You might need administrator permissions depending on your system configuration.
4

Restart the App

Launch Cheating Daddy again. The app will create fresh storage and you'll need to reconfigure your settings, including your API key and any custom preferences.

What gets reset:

  • API key configuration
  • App settings and preferences
  • Window position and size
  • Any cached data or temporary files
